# Image Placeholders

This folder contains placeholder images for the Delicious Bites website.

## Required Images

### Background Images
- **background-placeholder.jpg** - Used for hero sections and background decoration
  - Recommended size: 1920x1080px
  - Should have soft bakery theme colors (#F78CA2, #FFD6DF, #FFF7F9)

### Product Images  
- **product-placeholder.jpg** - Used when products don't have custom images
  - Recommended size: 600x600px
  - Should show generic bakery items (cupcakes, cakes, etc.)

## Usage

All images use `alt="pictures"` as requested. Replace these placeholder paths with actual bakery photos:

- Hero backgrounds
- Product card images
- Category banners
- About page imagery

## Image Guidelines

For best results, use:
- High-quality photos (minimum 1200px width)
- Soft lighting and pastel bakery colors
- Professional product photography
- Consistent styling across all images
